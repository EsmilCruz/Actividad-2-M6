Tabla en html Una tabla en HTML organiza datos en filas y columnas usando la etiqueta table. Dentro, las filas se definen con tr, las celdas con td (datos) y th (encabezados). Secciones como thead, tbody, y tfoot agrupan el contenido. Formulario en html El formulario en HTML permite recolectar datos ingresados por el usuario y enviarlos a un servidor para su procesamiento.
Etiqueta table La etiqueta "table" en HTML define una tabla que organiza datos en filas y columnas. Etiqueta from La etiqueta "form" define el contenedor de un formulario, que puede tener atributos como "action" para especificar dónde enviar los datos y "method" para definir el tipo de solicitud.
Atributos de table Los atributos de "table" permiten personalizar la apariencia y estructura de la tabla, como "border", "cellspacing" y "width". Etiqueta input La etiqueta "input" se utiliza para crear diferentes tipos de campos de entrada, como texto, contraseñas, botones y más.
Etiqueta tr La etiqueta "tr" representa una fila de la tabla, agrupando las celdas que se muestran en línea horizontal. Etiqueta label La etiqueta "label" se usa para asociar un texto descriptivo con un elemento de formulario, mejorando la accesibilidad.
Atributo tr El atributo de "tr" puede incluir estilos como "align" o "valign" para alinear el contenido de la fila. Etiqueta select La etiqueta "select" permite crear un menú desplegable con múltiples opciones para que el usuario seleccione una.
Etiqueta td y th Las etiquetas "td" y "th" definen celdas en una tabla, donde "td" es una celda normal y "th" es una celda de encabezado. Etiqueta textarea: La etiqueta "textarea" crea un campo de texto más grande que permite la entrada de varias líneas.
Atributos de td y th Los atributos de "td" y "th" permiten fusionar celdas usando "colspan" y "rowspan", y también aplicar estilos como "width" o "height". Etiqueta fieldset, legend La etiqueta "fieldset" agrupa elementos relacionados en un formulario, y "legend" añade un título a dicho grupo.
Tablas avanzadas Las tablas avanzadas incluyen funciones y características adicionales, como la capacidad de ordenar y filtrar datos. Etiqueta meter La etiqueta "meter" muestra un indicador gráfico de un valor dentro de un rango específico, como niveles de batería o puntuaciones.
Etiqueta thead La etiqueta "thead" agrupa la cabecera de la tabla, que generalmente contiene los títulos de las columnas. Etiqueta progress La etiqueta "progress" indica visualmente el progreso de una tarea, como la carga de un archivo.
Etiqueta tfoot La etiqueta "tfoot" se usa para definir el pie de una tabla, que puede contener totales o información resumida. Etiqueta datalist La etiqueta "datalist" proporciona una lista de opciones predefinidas que se pueden usar para completar un campo de entrada.
Etiqueta tbody La etiqueta "tbody" agrupa el cuerpo de la tabla, donde se encuentran los datos principales o el contenido de la tabla. Etiqueta summary La etiqueta "summary" se utiliza junto con la etiqueta "details" para proporcionar un resumen visible que el usuario puede hacer clic para revelar más información.


Tema 2. Define los elementos que componen la estructuración de una página web:

  1. Contenedor (wrapper)

    2. Es un elemento que envuelve todo el contenido de una página web. Su principal función es agrupar y organizar los diferentes elementos (cabecera, contenido, pie, etc.)
    en un bloque estructural, generalmente para facilitar el diseño y la disposición del contenido en la pantalla.


  2. Cabecera (header)

    3. Es la sección superior de una página web. Suele contener elementos importantes como el logotipo, el título de la página, los menús de navegación, y en algunos
    casos, información de contacto o enlaces a redes sociales. Es un componente clave para la identidad visual de la web.


  3. Contenido (content)
    4. Es el área principal de la página web donde se muestra la información relevante que el usuario está buscando. Puede incluir texto, imágenes, videos, formularios
    y cualquier otro tipo de información importante.

  4. Menú (menu)

    5. Es el conjunto de enlaces que facilitan la navegación a través de las diferentes secciones o páginas de un sitio web. Los menús pueden estar ubicados en la
    cabecera, en el pie, o incluso en un lateral de la página.


  5. Pie (footer)

    6. Es la sección inferior de una página web. En el pie de página se suelen incluir enlaces a términos de uso, políticas de privacidad, información de derechos de
    autor, enlaces de contacto, y otros elementos de navegación secundarios.


  6. Lateral (sidebar)

    7. Es una barra que se encuentra a uno de los lados del contenido principal. Suele incluir elementos adicionales como menús de navegación, widgets, enlaces a otros
    contenidos, anuncios, o información complementaria.


  7. Etiqueta div
    8. La etiqueta div es un contenedor genérico utilizado para agrupar otros elementos HTML. Se utiliza frecuentemente para estructurar y organizar el diseño de una página
    web mediante el uso de CSS para aplicar estilos o disposición específica.
